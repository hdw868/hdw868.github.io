Python module Fabric2

网上关于Python的fabric在运维中的文章也不少，不过发现大都是有些年头了。大部分例子使用的是早期的fabric1.x（适用于Python2）或者fabric3（适用于Python3），如果你使用直接使用pip install fabric而不指定版本的话，参照文章的代码运行时会发现from fabric.api import run之类的出错。我去官网查了下，发现作者将之前的fabric项目一分为二了，一个是类似于rake的invoke模块，另一个则是基于ssh的上层应用fabric模块（Python2和Python3都支持）。
